3.超链接

Markdown 支持两种形式的超链接： 行内式和参考式。 行内式一般作为主流使用。

3.1.行内式

语法格式 [Text1](Url "Text2")
Text1 直接显示文字：直接能看见的文字
Url 超链接url（http://www....）
Text2 鼠标悬停时显示的文字，该参数可选，与前面的Url中间要用空格隔开
